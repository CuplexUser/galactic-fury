# Galaxy Fury

Galaxy Fury is a retro vertical scrolling space shooter built with Pygame. Blast enemy squadrons, dodge swarms, and clear three escalating stages capped by boss fights.

## Requirements
- Python 3.9+
- Pygame

## Setup
1. Create and activate a virtual environment (optional).
2. Install dependencies:
   - `pip install pygame`
3. Run the game:
   - `python main.py`

## Controls
- Arrows: Move
- Space: Fire
- P: Pause
- ESC: Back (menus)

## Options
From the main menu, open **Options** to adjust SFX and Music volume.
